dstc2_results

- You may run the evaluation for yourself by using the run_evaluation.py script provided. This will require the dstc2 data which is available at http://camdial.org/~mh521/dstc/

- The featured metrics are tabulated in featured.csv, and the full list of metrics can be found in all.csv

- For each entry there is:
    - a .txt file, giving some more information about the approach used
    - a .test.json file, giving the output of the tracker on the test set
    - a .test.score.csv file, giving the scores on the test set
    - a .dev.score.csv file, giving the scores on the dev set
